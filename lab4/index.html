Table

In this lab, i made a table that shows student information, i used the following tags

  1. Table Tag- Used to create the structure of the table
  2. Caption Tag - i used it to give the title to the table
  3. thead Tag- i used it to define the table header section, it is semantic tag
  4. tr Tag- i used it to define rows in the table
  5. th Tag- i used it to define Table headings
  6. Tbody Tag - i used it to define content section of the table
  7. td Tag- it is used to represent table data,used inside rows

Form

i also made a form for a new student to register himself, i used the following tags

  1. form Tag- i used it to create the form
  2. fieldset Tag- i used it to group related form elements together and add a border around them.
  3. legend Tag - i used it to provide a title or caption for each fieldset.
  4. label Tag - i used it to define labels for the input fields to make them user-friendly and accessible.
  5. input Tag - i used it take different types of user input like text, number, radio buttons,and submit.
  6. select Tag - i used it to create a dropdown menu for selecting the semester number.
  7. option Tag - i used it inside the select tag to define each choice in the dropdown.
